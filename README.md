

# analise-vendas-qualidade-dados
Realizei análise de qualidade de dados, identificando 41% de inconsistências e analisando sua distribuição por dimensões como produto, cidade e forma de pagamento.

   Objetivos
Identificar erros na base de dados
Realizar limpeza e tratamento dos dados
Criar indicadores de qualidade (KPI)
Construir um dashboard para análise


  Tratamento de Dados

A base apresentava diversos problemas, como:

Valores inválidos (#ERRO, #VALOR, #NOME)
Dados faltantes ("Não informado")
Inconsistência em valores numéricos
Problemas de formatação

Após o tratamento, foi possível obter:

 100% de qualidade dos dados
 Redução de inconsistências (41,38% inicialmente)


  Dashboard

O dashboard apresenta:

Faturamento por produto
Faturamento por cidade
Análise por forma de pagamento
Indicadores de qualidade dos dados



   Insights
Alta presença de inconsistências na base original
Necessidade de tratamento para análises confiáveis
Concentração de faturamento em determinados produtos
Variação de receita entre cidades


   Ferramentas utilizadas
LibreOffice Calc
Tratamento manual de dados
Tabelas dinâmicas
Visualização de dados
